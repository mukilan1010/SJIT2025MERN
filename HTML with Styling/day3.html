Basic Git Commands & Syntax

Command Description Syntax
Initialize Git Repository Creates a new Git repository git init
Clone Repository Copies a remote repository to local git clone <repo_url>
Check Repository Status Shows changes and tracking status git status
Add Changes Adds files to staging area git add .(All changes)
git add <filename>(Specific file)
Commit Changes Records changes in the repository git commit -m "Commit message"
Push Changes Uploads local changes to remote git push origin main
Pull Changes Fetches updates from remote repository git pull origin main
